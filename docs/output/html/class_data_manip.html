Retrieves flights based on specified origin and destination parameters.

This method retrieves flights based on the origin and destination parameters, along with optional filters and radius constraints. It uses the graph to find paths between airports and considers different types of origin and destination specifications.

Parameters
originThe origin location or airport code.
destThe destination location or airport code.
oTypeThe type of origin specification (1: Airport Code, 2: City, 3: Closest Airport, 4: Near Location).
dTypeThe type of destination specification (1: Airport Code, 2: City, 3: Closest Airport, 4: Near Location).
filtersA vector of airline codes to filter the paths.
oRadiusThe radius for origin specification (default: 0 for Airport Code or City).
dRadiusThe radius for destination specification (default: 0 for Airport Code or City).
